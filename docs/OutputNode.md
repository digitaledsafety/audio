# Output Node

The Output Node is a final destination for audio signals. You can have multiple Output nodes in a workspace, each with its own volume and mute controls. All Output nodes are summed together at the global master gain stage.

## Inputs

*   **Audio In**: The audio input to the node.
*   **Gain CV**: A CV input to modulate the gain of this output node.

## Controls

*   **Gain**: Sets the local volume level for this output.
*   **Mute**: A toggle to silence this output without changing its gain setting.
